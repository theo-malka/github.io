Spotify data visualization project
Dashboard documentation

This dashboard was developped by Théo-Malka Lacombe, Maxime Peter and Julien Verdun as part of a data visualization class at Ecole Centrale de Lyon.

How can I visualize my own data ?

First you need to ask Spotify for your own personal data and then create Spotify Web Developper account, in order to use the Spotify API. The next step is to use the python notebook to extract your songs features. Then you will be able to upload those data with the button at the top left hand corner of the window and the visualizations will be updated with your personnal data. The documentation of the project is available here. Follow all the instructions in the README file and you will get your data to use for this dashboard.
